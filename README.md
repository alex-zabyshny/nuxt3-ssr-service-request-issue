## Install Dependencies

```bash
pnpm install
```

## Start Servers

### In the first terminal:

```bash
node server.mjs 3001
```

### In the second terminal:

```bash
node server.mjs 3002
```

### In the third terminal:

```bash
pnpm dev
```

## Open Browser

Navigate to:
[http://localhost:3000/email](http://localhost:3000/email)
